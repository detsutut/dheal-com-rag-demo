<!-- INSTALLATION -->
## Installation

1. Make sure you have the latest version of pip installed
    ```sh
    pip install --upgrade pip
    ```
2. Install dependencies from requirements
    ```sh
    pip install -r --no-cache-dir requirements.txt
    ```
3. Run the Gradio app
    ```sh
    python3 app.py
    ```
